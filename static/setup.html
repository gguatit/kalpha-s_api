한 줄 설치

아래 명령을 실행하면 서버에서 OS를 감지해 로컬 사용자 경로 또는 `/usr/local/bin`에 `kalpha` 실행 스텁을 설치합니다.

curl -sL https://api.kalpha.kr/setup | bash

대신 검증된 환경에서는 먼저 스크립트를 내려받아 내용 확인 후 실행하세요:

curl -sL https://api.kalpha.kr/setup -o setup.sh
less setup.sh
bash setup.sh

설치 결과

  • 기본적으로 `kalpha` 실행 파일을 설치합니다 (간단한 CLI 스텁).
  • 경로에 따라 `/usr/local/bin` 또는 `$HOME/.local/bin`에 설치됩니다.
  • 설치 후 `kalpha help` 또는 `kalpha version`으로 동작을 확인하세요.

보안 주의사항

  • 이 방식은 편리하지만 위험할 수 있습니다. 스크립트 파이프 실행은 신뢰할 수 있는 출처에서만 사용하세요.
  • 가능하면 스크립트를 먼저 내려받아 내용과 체크섬을 확인한 뒤 실행하세요.

문제 해결

  • 설치 후 `kalpha`가 인식되지 않으면 경로가 설정되지 않은 것입니다. 아래를 쉘에 추가하세요:
    • export PATH="$HOME/.local/bin:$PATH"
  • 권한 문제로 `/usr/local/bin`에 설치되지 않는 경우, 스크립트가 자동으로 `$HOME/.local/bin`에 설치합니다.
